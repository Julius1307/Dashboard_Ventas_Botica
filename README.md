# Dashboard_Ventas_Botica
Dashboard Dinámico de Ventas en Excel (Tablas Dinámicas/Slicers) para análisis de negocio e inventario. Monitorea KPIs clave: Ventas Totales, Producto más vendido, Stock Crítico y Tendencia Mensual. Solución automatizada con VBA para el registro de datos y la exportación instantánea del reporte a PDF con un clic.
